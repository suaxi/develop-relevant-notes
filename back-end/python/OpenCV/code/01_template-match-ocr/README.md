### ocr 识别流程

*PyCharm env环境选择 Python 3.6, opencv-python 3.4.1.15*

1. 读取（导入）模板
   - 灰度处理
   - 二值处理

2. 绘制轮廓（外）
3. 轮廓排序（分离轮廓并按顺序存储）
4. 读取要识别的银行卡
5. 将银行卡（源图）resize并灰度处理
6. 礼帽操作
7. Sobel算子
8. 闭运算（先膨胀，再腐蚀），突出要识别的区域
9. 二值处理
10. 绘制要识别区域的外轮廓
11. 提取数字轮廓
    - 绘制外接矩阵
    - 二值处理
    - 与模板数字一一比对（score）
12. 输出结果
